
        Bunga Api

        tahun: 2014
        soundcloud: https://soundcloud.com/amirmeludah/bunga-api
        lirik:

        bunga api dibakar malam ini
mari rai bebas perintah koloni
merdeka asas tunjang hidup harmoni
keris dan pedang disimpan dalam laci
autoriti
patriarki
tiada insan berani lontar batu api
katak bawah tempurung akhirnya kan mati
tanpa informasi
kita kan ditepi
malaysia malaya tanah melayu
tempat tumpah darah datuk nenek moyang aku
tak perlu bersatu
sama sang penipu
cuma ganding bahu slogan syahdu
duduk lepak di gerai makan
roti canai ditepuk
empuk
sebelum ditelan
sembang kuat
dengan rakan-rakan
bila depan orang lain terbisu jangan
oh jangan bakar semangat
akukan beri kau tiga harakat
untuk lari dari aku dengan cepat
sakratul maut jika kau terhambat
cintaku pada sekeping tanah
tanya si anak kenapa kita perlu berdarah
"hai sayang tidak boleh berserah
bila ketua tidur dengan pihak salah"
